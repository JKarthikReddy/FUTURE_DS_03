# 🎓 College Event Feedback Analysis

## 📊 Overview  
This project analyzes student feedback from college events (TechFest, Workshops, Seminars, Cultural Programs) using **Python, pandas, and NLP** techniques.  
The goal is to extract insights about **satisfaction, engagement, and improvements** to help organizers plan better events.

---

## 🎯 Objectives  
- Clean and transform raw feedback data.  
- Analyze student ratings and engagement across events.  
- Perform **sentiment analysis** on open-text feedback.  
- Visualize results using bar charts, pie charts, and word clouds.  
- Provide actionable recommendations for organizers.  

---

## 🚀 Features  
- **Data Cleaning** with pandas  
- **EDA**: Ratings distribution, event popularity, engagement trends  
- **Sentiment Analysis** with TextBlob  
- **Word Clouds** for positive comments and improvement areas  
- **Charts & Dashboard Preview** (ratings, sentiment, engagement)  

---

## 🛠 Tools & Libraries  
- Google Colab / Jupyter Notebook  
- Python (pandas, numpy)  
- matplotlib, seaborn  
- TextBlob (sentiment analysis)  
- wordcloud  

---

## 📈 Key Insights  
- ⭐ **TechFest** had the highest ratings and engagement.  
- 😊 60% of comments were positive, describing events as *fun* and *informative*.  
- ⚠️ Improvement areas: **time management & scheduling**.  

---

## 💡 Recommendations  
- Prioritize highly engaging events like **TechFest**.  
- Improve **event scheduling and coordination**.  
- Maintain **interactive formats** to boost participation.
